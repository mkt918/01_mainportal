# 📚 授業記録システムの使い方ガイド

このガイドでは、Markdown形式で授業記録を簡単に作成する方法を説明します。

## 🎯 このシステムでできること

- **簡単な記録作成**: テンプレートをコピーして穴埋めするだけ
- **3択クイズ**: 前回の復習とまとめクイズを自動で対話型に変換
- **自動更新**: ボタン1つでポータルサイトに反映

---

## 📝 授業記録を作成する手順

### ステップ1: テンプレートをコピー

1. `note/lessons/template.md` を探します
2. ファイルを右クリック → **コピー**
3. 同じフォルダ内で右クリック → **貼り付け**
4. コピーされたファイルの名前を変更します

**ファイル名の付け方**:

```
YYYY-MM-DD-科目名.md
```

**例**:

- `2024-02-05-簿記.md`
- `2024-02-06-情報処理.md`
- `2024-02-07-プログラミング.md`

---

### ステップ2: 内容を編集

1. 作成したファイルを**メモ帳**または**VSCode**で開きます
   - 右クリック → **プログラムから開く** → **メモ帳**

2. 以下の項目を穴埋めします:

```markdown
---
date: 2024-02-05          ← 授業の日付
subject: 簿記             ← 科目名
period: 1                 ← 時限（1, 2, 3...）
unit: 仕訳の基本          ← その日の単元名
---
```

1. **前回の復習**を記入:

```markdown
## 📚 前回の復習

**あらすじ**: 前回の内容を1-2行で要約

### 復習クイズ

**Q1**: 問題文？
- A) 選択肢1
- B) 選択肢2 ✓    ← 正解に✓を付ける
- C) 選択肢3
```

1. **本日の予定**を記入:

```markdown
## 📅 本日の予定

1. 今日やること1
2. 今日やること2
3. 今日やること3
```

1. **資料**を記入（任意）:

```markdown
## 📎 資料

- [スライド](./slides/2024-02-05.pdf)
- [配布資料](./materials/handout.pdf)
```

※資料がない場合はこのセクションを削除してOK

1. **まとめクイズ**を記入:

```markdown
## ✅ まとめクイズ

**Q1**: 問題文？
- A) 選択肢1
- B) 選択肢2 ✓    ← 正解に✓を付ける
- C) 選択肢3
```

1. **保存**します（Ctrl + S）

---

### ステップ3: HTMLに変換

1. プロジェクトのフォルダ（`01_mainportal`）を開きます
2. **`update-lessons.bat`** をダブルクリック
3. 黒い画面が表示され、自動で変換が実行されます
4. 「処理が完了しました！」と表示されたら **Enter** を押します

---

### ステップ4: 確認

1. ブラウザで `index.html` を開きます
2. 「授業の記録」セクションに新しい記録が表示されているか確認
3. クリックして内容を確認
4. クイズが正しく動作するか試してみましょう

## 💡 よくある質問

### Q: メモ帳で開くと文字化けします

**A**: ファイルの文字コードが「UTF-8」になっているか確認してください。VSCodeの使用をおすすめします。

### Q: クイズの正解はどうやって指定しますか？

**A**: 正解の選択肢の後ろに `✓` マークを付けます。

```markdown
- A) 選択肢1
- B) 選択肢2 ✓  ← これが正解
- C) 選択肢3
```

### Q: 資料のリンクはどうやって書きますか？

**A**: 以下の形式で記述します:

```markdown
[表示名](ファイルへのパス)
```

例: `[スライド](./slides/2024-02-05.pdf)`

### Q: 授業記録を削除したい

**A**: `note/lessons/` から該当する `.md` ファイルを削除し、再度 `update-lessons.bat` を実行してください。

### Q: エラーが出ました

**A**: 以下を確認してください:

1. Node.jsがインストールされているか
2. `npm install` を実行したか
3. Markdownの書式が正しいか（特に `---` の位置）

## 🔧 初回セットアップ（1回だけ）

### Node.jsのインストール

1. [Node.js公式サイト](https://nodejs.org/)にアクセス
2. 「LTS」版をダウンロード
3. インストーラーを実行（すべてデフォルトでOK）

### 依存関係のインストール

1. コマンドプロンプトを開く
2. プロジェクトフォルダに移動:

```
cd C:\Program2\T\01_mainportal
```

3. 以下を実行:

```
npm install
```

これで準備完了です！

## 📌 チェックリスト

授業記録を作成する際のチェックリストです:

- [ ] テンプレートをコピーして日付・科目名でリネーム
- [ ] 日付、科目、時限、単元を記入
- [ ] 前回の復習（あらすじ + クイズ2問）を記入
- [ ] 本日の予定を記入
- [ ] 資料リンクを記入（任意）
- [ ] まとめクイズ（2問以上）を記入
- [ ] 正解に ✓ マークを付ける
- [ ] ファイルを保存
- [ ] `update-lessons.bat` を実行
- [ ] ブラウザで確認

## 🎨 カスタマイズ

### クイズの数を増やす

Q3, Q4... と自由に追加できます:

```markdown
**Q3**: 問題文？
- A) 選択肢1
- B) 選択肢2 ✓
- C) 選択肢3
```

### セクションを追加

Markdownの見出し（`##`）を使って自由にセクションを追加できます:

```markdown
## 📝 補足説明

ここに補足内容を書く
```
